generated

fun generated(generatorName: String, generatedAt: Instant? = null, comments: String = "", targetVersion: JavaVersion = JavaVersion.VERSION_1_9): AnnotationSpec

Build an annotation that marks a type as generated.

The resulting annotation is javax.annotation.Generated for java versions <= 8 and javax.annotation.processing.Generated for java versions >= 9.

Parameters

generatorName - Fully qualified name of the code generator.

generatedAt - Date and time when the code was generated.

comments - Any comments that the code generator may want to include in the generated code.

targetVersion - Java version to generate the annotation for
